
        Задание:
            Дано бинарное дерево. Определить количество узлов на N-м 
            уровне бинарного дерева (N ввести с клавиатуры).
            
            
        Инструкция:
        1.Запустите файл: 
            gcc -Wall -Wextra -o lab4.exe lab4.c
            ./lab4.exe
        2.На экране появится предложение ввести первый элемент дерева (корень).
        Введите его.Далее появится предложение ввести очереденой элемент дерева.
        Введите его.Предложения буду выводиться на экран до тех пор, пока вы не введёте "0".
        Далее появится предложение ввести уровень, для которого нужно расчитать кол-во узлов.
        Введите его.
        3.На экране отобразится количество узлов на данном уровне либо "0", если данного уровня
        не существует либо на уровне нет узлов.
